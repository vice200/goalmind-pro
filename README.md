# GOALMIND PRO – Football Predictions (Poisson + AI + Kelly)

GOALMIND PRO is a multi-league football prediction system combining:

✔ Poisson probability  
✔ Dixon–Coles correction  
✔ AI (Random Forest) for FT 1X2  
✔ AI for Goals (Over/Under 2.5, BTTS, Expected Goals)  
✔ Kelly staking  
✔ Value bets detection  
✔ Excel PRO Export  
✔ Login protection  

This is a production-grade Streamlit dashboard ready for analysts or paid subscribers.

### Features
- 20+ leagues (Premier League, Championship, League 1–2, Bundesliga, Ligue 1–2...)
- Auto-updated from Football-Data.co.uk
- AI models trained on multi-season dataset
- Fixtures module + value bets
- Export to Excel PRO mini-app
- Clean modern UI with KPIs and charts
